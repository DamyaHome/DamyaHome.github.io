  1. vue-cli的简介与使用
    官网 –》学习 –》教程 –》安装 –》命令行工具

    node -v //node.js官方安装
    npm -v
    npm install –global vue-cli //全局安装
    vue init webpack my-project //创建一个基于webpack模板的新项目
    cd my-project //安装依赖
    npm run dev

build：项目webtate配置文件
config：开发环境线上环境配置文件
node——modules：依赖
src：源代码（可以使用es6语法）

main.js:挂载文件
xxx.vue:单文件组件（内容清空）
static：静态资源
