安装

常用操作

new 里按照安装的内核可以新建对应语言的notebook

notebook 界面由以下部分组成：

  1. notebook 的名称
  2. 主工具栏，提供了保存、导出、重载 notebook，以及重启内核等选项
  3. 快捷键
  4. notebook 主要区域，包含了 notebook 的内容编辑区

每个 notebook 由多个单元格构成，而每个单元格又可以有不同的用途

比如在python的notebook里新建一个cell,可以是代码cell,也可以是md语言的cell,适当搭配可以使页面美观

Jupyter Notebook 支持修改之前的单元格而使整个代码发生改变,而每个单元格又可以单独执行和计算

MarkDown 单元格支持HTML语法,也支持可以插入数学公式的 LaTex 语法

巧用命令

Cell -> Run all 可以重新计算整个notebook
Insert -> Insert单元格above 在上方插入单元格
Edit -> Delete Cell 删除某个cell(单元格)
Edit -> Move cell up/down 移动某个单元格
Edit -> Cut Cell 剪切cell
Edit -> Paste Cell Above/Below 粘贴cell
Edit -> Merge Cell Above/below 如果notebook中有很多cell只需要执行一次，或者想一次性执行大段代码，那么可以选择合并这些cell

导出功能

Jupyter Notebook 导出文件时可以选择不同格式

比如:
HTML
Markdown
ReST
PDF（通过 LaTeX）
Raw Python

导出 PDF 功能，可以不用写 LaTex 即可创建漂亮的 PDF 文档

还可以将 notebook 作为网页发布在网站

甚至可以导出为 ReST 格式，作为软件库的文档
