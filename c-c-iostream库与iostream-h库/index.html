iostream 库

c++中为了避免名字定义冲突,引入了 namespace(名字空间) 定义,在使用的时候输出需要引用 std::cout<<x;

引入 std:: 有三种方法:

  1. using namespace std; cout<<x;

  2. using std::cout; cout<<x;

  3. std::cout<<x;

可见第一种方法最为便捷

iostream.h 库

继承了c语言的标准库文件,没有引入名字空间定义,当代码中使用了<iostream.h>库时,输出可直接引用 cout<<x;
